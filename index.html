    Project maintained by zeroproyect Hosted on GitHub Pages — Theme by mattgraham

    Bienvenido a MultiForum Open Script.

    MultiForum Open Script se empezo a desarrollar en 2010 con la idea de hacerlo privado el codigo , despues de la lamentable separacion del equipo Tecnico de Host Sky y del cese de los servicios de Dicha empresa el script se quedo en el olvido . Hasta que el C.O.M.A del equipo Español de Host Sky lo rescato y empezo a desarrollarlo .

    El Proposito de esta GitHub es hacer un codigo abierto y desarrollar el script siguiendo las siguiente etapas antes de sacar una version :

    > Version Beta
> Version RC
> Version Oficial

    Siempre aplicando las ideas de nuestros usuarios

    Licencia de MultiForum Open Script

    La licencia actual de MultiForum Open Script es de distribucion gratuita es decir no contamos con la licencia aun legalizada pero sin dudar seria una licencia de Desarrollo y Codigo Libre.
